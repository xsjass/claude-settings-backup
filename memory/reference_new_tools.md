**MCP Servers (configured in ~/.mcp.json):**
- **Context7** (`@upstash/context7-mcp`) — Fetches current library docs into context. Use for Next.js/Prisma/Tailwind.
- **GitHub** (`@modelcontextprotocol/server-github`) — Native GitHub repo/PR/issue control.
- **Playwright** (`@playwright/mcp`) — Browser automation for testing and scraping.

**Repos cloned:**
- **System Prompts** (`~/.claude/system-prompts/`) — 128K stars, leaked prompts from Cursor, Devin, v0, Windsurf. Study for prompt engineering.
- **PR Agent** (`~/.claude/pr-agent/`) — 10.5K stars, AI PR reviewer. Semantic code review.

**How to apply:** Use Context7 when generating framework code (pulls current docs). Use Playwright MCP for testing web apps. Reference system-prompts for improving CLAUDE.md and agent prompts. Use PR Agent patterns for code review.
