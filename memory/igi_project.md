Project IGI: "You bring the sounds. We make the beat."
Located at E:/PROJECT IGI/

3-Layer Model: Sound Library → AI Arrangement Engine → Full DAW
User picks sounds, AI arranges them into professional beats, DAW to edit everything.

**Current Phase:** NO DAW. Pure "ingredient → AI cooks → beat" flow.

**Tech Stack:** React/Next.js + TypeScript, WebAudio API

**Genres:** Trap, Melodic Trap, Drill (UK/Chicago/Brooklyn), Boom Bap, Lo-Fi, Phonk, RnB, Afrobeat, Jersey Club

**Concept:** Like a kitchen — user brings ingredients (sounds), AI cooks the beat. No editing, no DAW.

**Key Features (44 files, zero errors, 3 routes):**
- Melody-first CookEngine v3 (1688 lines) — melody drives everything, drums serve melody
- 12 genres including Hip-Hop
- 105 creative effects across 5 categories (rhythm, texture, arrangement, vibe, production)
- 25 type beat presets (Travis Scott, Drake, Metro Boomin, etc.)
- 90+ ingredient categories, unlimited layers
- Smart Suggestions engine — AI recommends missing ingredients
- Mood Board — type "dark aggressive trap" and AI maps to settings
- Beat DNA Visualizer — visual fingerprint of the cooked beat
- Beat Rating & Feedback system
- Real WebAudio playback (BeatPlayer.ts)
- Real WAV export with stems (ExportEngine.ts + wavEncoder.ts)
- Sound upload with preview and auto-categorization
- Landing page with hero, feature pills, how-it-works
- Custom duration (any length 30s-5min)
- Toast notifications, keyboard shortcuts, PWA manifest
- Premium dark UI with framer-motion animations, glassmorphism
