Francify: AI-powered French learning website
Location: C:/Users/xsjas/Desktop/Francify/

**Tech Stack:**
- Next.js 14 App Router + TypeScript
- Prisma + PostgreSQL
- Tailwind CSS + Framer Motion (heavy animations)
- Zustand (state management)
- NextAuth.js (authentication)
- OpenAI API (AI tutor, pronunciation, grammar)
- Stripe (payments)
- Socket.io (real-time features)

**Core Features:**
- CEFR A1-C2 structured courses
- Spaced Repetition System (SM-2 algorithm)
- AI conversation partner
- Pronunciation scoring
- Gamification: XP, streaks, hearts, gems, leagues, achievements
- Story-based learning
- Grammar reference
- Vocabulary builder
- Daily challenges
- Leaderboards with weekly leagues

**Monetization:**
- Free tier (limited hearts, basic content)
- Pro: $9.99/mo (unlimited hearts, all content, AI tutor)
- Family: $14.99/mo (up to 5 accounts)

**Why:** JJ wants to build a premium French learning platform that mixes the best of all competitors with addictive algorithms and premium animations.
**How to apply:** Build with scale in mind, premium UI/UX, every interaction animated, dark theme with French blue/red accents.
